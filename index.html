Svátky API

URL

https://svatky.adresa.info/formát

Formát

json, xml, txt

HTTP metoda

GET

Parametry

Parametry jsou vyhodnocovány podle priority name > date.

name
jméno (case sensitive)
date
datum ve formátu DDMM
lang
cs || sk

Odpověď

json

[{"date": "datum", "name": "jméno"}...]

xml


<?xml version="1.0" encoding="UTF-8"?>
<svatky>
    <svatek>
        <date>datum</date>
        <name>jmeno</name>
    </svatek>
    ...
</svatky>

txt

datum;jméno
...

Příklady

  • https://svatky.adresa.info/txt?name=Petr
  • https://svatky.adresa.info/json?date=2202
  • https://svatky.adresa.info/xml?lang=cs
